1. Dézippez-le (clic droit → « Extraire tout », ou 7-Zip).

2. Lancez Relay-Setup.exe.

⚠ Si Windows affiche « Windows a protégé votre ordinateur » : clic droit sur le fichier → Propriétés → cochez « Débloquer » → OK (ou « Informations complémentaires » → « Exécuter quand même »). L'app n'est pas signée, mais elle est sûre.
