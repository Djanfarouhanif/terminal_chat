Documentation

Tout ce qu'il faut pour installer et utiliser Relay.

1. Installation

Relay s'installe en quelques secondes, sans dépendance à gérer.

  1. Télécharge l'archive depuis la page de téléchargement (c'est un .zip).
  2. Dézippe-la (clic droit → « Extraire tout », ou 7-Zip).
  3. Lance Relay-Setup.exe et suis l'assistant (installation par utilisateur, sans droits administrateur).
  4. La commande relay devient disponible dans n'importe quel terminal, et un raccourci est ajouté au menu Démarrer et au bureau.

⚠ Si Windows affiche « Windows a protégé votre ordinateur » : clic droit sur le fichier → Propriétés → coche « Débloquer » → OK (ou « Informations complémentaires » → « Exécuter quand même »). L'application n'est pas signée mais elle est sûre.

2. Lancer Relay

┌─you@relay:~ └─$ relay

Ou double-clique sur le raccourci Relay. Au démarrage, la liste des commandes s'affiche.

3. Créer un compte & se connecter

Au premier lancement, crée ton compte (pseudo, email, mot de passe) :

/register monpseudo mon@email.fr monMotDePasse

Pour te reconnecter plus tard :

/login monpseudo monMotDePasse

🔒 Astuce : tape simplement /login (sans rien d'autre) pour saisir ton mot de passe masqué, à l'abri des regards.

Tes identifiants sont mémorisés : aux lancements suivants, tu es reconnecté automatiquement — pas besoin de retaper quoi que ce soit.

4. Discuter en groupe (salons)

Un salon est une discussion à plusieurs, en temps réel.

/channels # voir les salons existants /create backend # créer un salon /join backend # rejoindre / ouvrir un salon Salut l'équipe ! # écrire : tape du texte SANS « / » /leave # quitter le salon

Tout le monde présent dans le salon voit tes messages instantanément. Ceux qui rejoignent plus tard retrouvent l'historique. Tu peux corriger ou retirer tes messages avec /edit <id> et /del <id> (chaque message affiche son #id).

5. Messages privés (DM)

Pour parler à une seule personne, en privé et en temps réel :

/dm alice Salut, tu es dispo ? # envoyer un MP à alice /dm alice # revoir l'historique avec alice /users # voir qui a un compte

Quand quelqu'un t'écrit en privé, le message apparaît aussitôt dans ton flux — ✉ alice (privé)> … — avec une petite notification sonore, même si tu es dans un salon. Personne d'autre ne voit vos échanges.

6. Toutes les commandes

/helpAfficher l'aide
/channelsLister les salons
/create <nom>Créer un salon
/join <nom>Rejoindre / ouvrir un salon
/leaveQuitter le salon courant
/users · /onlineLister les utilisateurs
/dm <user> [msg]Message privé / historique
/historyRecharger l'historique
/search <texte>Rechercher partout
/edit <id> <texte>Éditer un de ses messages
/del <id>Supprimer un de ses messages
/profile · /status <s>Profil · statut (online/busy/away/offline)
/whoami · /clearInfos session · effacer l'écran
/logout · /exitSe déconnecter · quitter

7. Raccourcis clavier

Ctrl + NNouveau salon
Ctrl + KRecherche
Ctrl + LEffacer l'écran
Ctrl + QQuitter

8. Mises à jour

Relay vérifie au démarrage si une version plus récente est disponible et te prévient avec le lien de téléchargement. Il te suffit alors de récupérer le nouvel installeur sur la page de téléchargement et de le relancer.
