Documentation

Tout ce qu'il faut pour installer et utiliser Relay.

1. Installation

Relay s'installe en quelques secondes, sans dépendance à gérer.

  1. Télécharge l'installeur depuis la page de téléchargement.
  2. Lance Relay-Setup.exe et suis l'assistant (installation par utilisateur, sans droits administrateur).
  3. La commande relay devient disponible dans n'importe quel terminal, et un raccourci est ajouté au menu Démarrer et au bureau.

⚠ L'installeur n'est pas signé : Windows peut afficher « Windows a protégé votre ordinateur ». Clique sur « Informations complémentaires » → « Exécuter quand même ».

2. Lancer Relay

┌─you@relay:~ └─$ relay

Ou double-clique sur le raccourci Relay. Au démarrage, la liste des commandes s'affiche.

3. Créer un compte / se connecter

/register monpseudo mon@email.fr monMotDePasse /login monpseudo monMotDePasse

Les identifiants sont mémorisés : aux lancements suivants, tu es reconnecté automatiquement. Le mot de passe peut être saisi masqué en tapant simplement /login puis en répondant aux invites.

4. Discuter

/join general # rejoindre un salon Salut tout le monde ! # message (texte sans « / ») /dm amin coucou # message privé

5. Commandes

/helpAfficher l'aide
/channelsLister les salons
/create <nom>Créer un salon
/join <nom>Rejoindre / ouvrir un salon
/leaveQuitter le salon courant
/users · /onlineLister les utilisateurs
/dm <user> [msg]Message privé / historique
/historyRecharger l'historique
/search <texte>Rechercher partout
/edit <id> <texte>Éditer un de ses messages
/del <id>Supprimer un de ses messages
/profile · /status <s>Profil · statut (online/busy/away/offline)
/whoami · /clearInfos session · effacer l'écran
/logout · /exitSe déconnecter · quitter

6. Raccourcis clavier

Ctrl + NNouveau salon
Ctrl + KRecherche
Ctrl + LEffacer l'écran
Ctrl + QQuitter

7. Mises à jour

Relay vérifie au démarrage si une version plus récente est disponible et te prévient avec le lien de téléchargement. Il te suffit alors de récupérer le nouvel installeur sur la page de téléchargement et de le relancer.
